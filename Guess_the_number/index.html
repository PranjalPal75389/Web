Guess the Number

Guess the number within the specified range.

To play, set the range of numbers and guess a number within that range. If the guess is incorrect, the game will provide a hint. Keep guessing until you correctly guess the number or reach the maximum number of attempts. The best score is 1, so try to beat it!

Rules

  1. Guess the Number between 1 to 100.
  2. Less attempt means better score.
